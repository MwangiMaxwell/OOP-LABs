 - Download dbfxapp folder from elearning

 - Unzip the folder to your desktop

 - Compile all java files, remember you have to cd into the folder and also it contains JavaFX code e.g. for myself I compiled using javac --module-path "C:\Program Files\Java\javafx-sdk-26.0.1\lib" --add-modules javafx.controls,javafx.fxml *.java
